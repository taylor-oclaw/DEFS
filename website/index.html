Built for the Future of Data

A filesystem designed from the ground up for AI workloads, semantic understanding, and extreme performance.

Content-Addressable Storage

Every block is hashed with BLAKE3. Automatic deduplication eliminates redundant data across your entire volume.

Semantic Relationships

Gravity bonds connect files by meaning, not just location. Your data organizes itself around intent.

CoW Snapshots

Lightweight copy-on-write versioning. Snapshots are instant, space-efficient, and fully traversable.

POSIX via FUSE

Mount DEFS as a normal filesystem. Applications use it transparently — no rewrites, no compatibility layers.

Crash-Safe WAL

Write-ahead logging guarantees consistency. Power loss, crashes, or kills — your data stays intact.

Async Backend

Fully async, tokio-compatible API. Built for modern Rust concurrency patterns from day one.

Quick Start

Install the CLI and create your first DEFS volume in under a minute.

bash — defs-cli
$ cargo install defs-cli --features std
$ defs mkfs my-volume.defs --label "My Data"
$ defs mkdir my-volume.defs /projects
$ defs write my-volume.defs /projects/hello.txt --data "Hello, DEFS!"
